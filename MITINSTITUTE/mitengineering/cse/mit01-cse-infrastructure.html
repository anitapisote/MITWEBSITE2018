Infrastructure
Computer Hardware And Microprocessor Lab

Objectives :The objective of the Computer Hardware & Microprocessor Laboratory (CHML) is to provide students an in-depth microprocessor-based design experience. The laboratory is equipped with Microprocessor Kits and Computer Systems.

Computer Programming Lab

Objectives : The objective of this lab is to provide the students hands on sessions on various programming technologies and tools for Android Development, .Net application development etc
Computer Software Lab

Objectives :The objective of this laboratory is to introduce and work in Image Processing Tools like MATLAB, SCiLab and also open source platforms like Unix and Linux.

Computer Networks Lab

Objectives :The objective of this Lab is to introduce the students to the configuration of network and networking devices. Students are able to design and implement network algorithms and protocols using open network simulation tool like CISCO packet tracer and programming languages like C, JAVA etc.
Project Lab

OBJECTIVE OF LAB : The objective of this Lab is to provide the students with the environment for project development.
Computer Design Lab

Objectives : The objective of this Lab is to provide the students knowledge of various designing aspects like compiler design, data structures and algorithm design for different problems. Practical sessions includes Data Structure, Design and Analysis of Algorithms and Principles of Compiler Design.
Computer Graphics Lab

Objectives : The objective of this lab is to support development of programs involving graphics. The lab is equipped with various graphical tools and software. These tools help to represent and manipulate visual and geometric information.
Database and Information Lab

Objectives :The objective of this lab is to develop the ability in the students to design, implement and manipulate databases , to understand the concepts of Analytical Processing (OLAP) and compare with Transaction Processing (OLTP) and to implement the algorithms used for various types of Data Mining Problems.
Software Development Lab

Objectives : The objective of this lab is to develop the skills in students for Big Data techniques and cloud computing concepts.
Computer Application Lab

Objectives :The objective of this lab to develop the software which are based on various open source technologies. Software development includes networking projects, Android applications and network security.
System Software Lab

Objectives :The objective of this lab is to focus a computer internal functioning, chiefly through an operating system to perform commands for Unix and execute shell scripts.
RESEARCH LAB 1 and 2

Objectives :To motivate the students to explore the innovative ideas
To pursue upcoming technologies,research and development in Computer Science.

Online Computing Center

Objectives : To enhance interest of students in learning
To improve teaching quality of faculty members
To promote self-learning habit among the students
To bridge the curricular gaps through other alternatives
